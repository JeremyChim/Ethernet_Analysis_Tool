# 以太网数据处理工具

这是一个将以太网原始数据转化为udp格式的脚本工具，用于源数据的转换，方便后期的数据分析。

## 使用方法 
1. 删除cache文件夹下的文件
2. 删除csv文件夹下的文件
3. 把日志文件命名为 log.txt 放在根目录
4. 运行app.py
5. 查看csv文件夹
